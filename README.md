## stylelint-config-zc

通过 npm 包的方式给你的项目引入 stylelint 规则

## 安装依赖

```
npm install stylelint-config-zc --save-dev
```

## 使用

在 package.json 里新增字段 stylelint。（也可以新建文件 stylelintrc）

```js
"stylelint": {
    "root": true,
    "extends": "stylelint-config-zc"
}
```
